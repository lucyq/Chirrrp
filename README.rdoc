
Chirrrp has been created as part of a workshop on Ruby on Rails.
It is a simple version of twitter in which users can register accounts and post "Chirps".
Users may also follow others. Messages are displayed in a newsfeed.

 

Ruby version: 2.1.3
Uses the following: devise, simple_form, carrierwave, twitter-text, sqlite3
